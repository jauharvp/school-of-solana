# school-of-solana

A collection of projects and tasks for learning Solana blockchain development.

## Task 1
Introduction to Solana basics and ecosystem.  
Complete Rust functions with answers to Solana concept questions.

## Task 2
Learn Rust programming with geometric shapes and calculator implementations.  
Implement structs, traits, and error handling for validation and operations.

## Task 3
Build an on-chain vault program using Anchor framework.  
Implement deposit, withdraw, and toggle lock instructions with PDAs and CPIs.

## Task 4
Develop a decentralized Twitter program on Solana.  
Add features for tweets, reactions, comments, and user interactions.

## Program Task
Create a custom dApp with Solana program and frontend.  
Deploy Anchor program with PDA, write tests, and build interactive UI.
